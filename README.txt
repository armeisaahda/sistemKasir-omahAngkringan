Selamat datang di README Website "Sistem Kasir Kafe Omah Angkringan"

==================================================================================================================================================================================
Introduction :

Website sistem kasir kafe omah angkringan merupakan yang berfungsi untuk melakukan pemesanan makanan dan
pembayaran ditambah dengan laporan keuangan. Website ini memiliki 2 user yaitu admin dan kasir. 

==================================================================================================================================================================================
Menu (Admin) :
1. Dashboard 
Sebagai introduction tentang website ini. Menampilkan berbagai informasi penting

2. Kasir
Untuk melakukan transaksi penjualan antara kasir dengan pelanggan

3. Daftar Menu
Berfungsi untuk menambahkan menu yang ada di Kafe Omah Angkringan beserta dengan detailnya. 

4. User 
Berfungsi untuk mengelola user yang bisa mengakses website ini. 

5. Laporan penjualan
Berfungsi untuk menampilkan laporan penjualan milik Kafe Omah Angkringan sebagai media informasi
tentang penjualan kafe ini. 

-----------------------------------------------------------------------------------------------------------------
Menu (Kasir) : 
Kasir hanya dapat mengakses dua menu yaitu sebagai berikut. 

1. Dashboard 
Sebagai introduction tentang website ini. Menampilkan berbagai informasi penting

2. Kasir
Untuk melakukan transaksi penjualan antara kasir dengan pelanggan

==============================================================================================================
Penggunaan Website

1. Untuk melakukan transaksi pada Kafe Omah Angkringan
2. Untuk melihat laporan keuangan Kafe Omah Angkringan
